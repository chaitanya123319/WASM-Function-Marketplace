# WASM-Function-Marketplace
A full-stack, containerized, decentralized Functions-as-a-Service (FaaS) platform where developers monetize WebAssembly binaries, consumers pay-per-call using credits, and contributor nodes execute WASM in sandboxed environments. All binaries are stored on IPFS via Pinata (no S3

# WASM Function Marketplace

## 📌 Overview
A platform to upload, manage and execute WebAssembly functions.

## ⚙️ Tech Stack
- Backend: Python (FastAPI)
- Frontend: React / Node.js
- Contributor Node: Go
- WASM Runtime

## 🚀 Features
- User authentication
- Upload WASM functions
- Execute functions securely
- API-based microservices

## ▶️ How to Run
# backend
cd backend
pip install -r requirements.txt
python main.py

# frontend
cd frontend
npm install
npm start
